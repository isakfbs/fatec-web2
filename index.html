Node.js

Como iniciar um servidor Node.js

Objetivo da Aula

O objetivo desta aula foi introduzir os conceitos básicos do Node.js, focando na criação de um servidor HTTP simples. Os alunos aprenderam como configurar um ambiente Node.js, criar um servidor básico e servir conteúdo estático (como um arquivo HTML). Além disso, foram abordados conceitos como manipulação de requisições, leitura de arquivos e configuração de variáveis de ambiente.

Tópicos abordados

  • O que é Node.js e por que é amplamente utilizado.
  • Diferenças entre Node.js e linguagens de programação tradicionais do lado do servidor.
  • Instalação do Node.js e NPM (Node Package Manager)
  • Criação de um projeto Node.js básico (package.json).

Exemplo Prático:

  • Durante a aula, os alunos implementaram um servidor HTTP simples que:
  • Escuta requisições em uma porta específica (padrão: 3000).
  • Serve um arquivo HTML (index.html) como resposta.
  • Serve um arquivo HTML (index.html) como resposta.
  • Loga detalhes das requisições (método, URL, cabeçalhos) no console.
